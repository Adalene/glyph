API Reference

GLYPH exposes a static JSON API for programmatic access. AI agents can fetch the full icon library, filter by category, or retrieve individual icon SVG strings.

GET/api/icons.json

Returns all icons with metadata and SVG paths. Perfect for AI agent tool calls.

fetch('/api/icons.json') .then(r => r.json()) .then(data => { // data.icons = [{id, name, category, svg, tags}, ...] // data.categories = ['ui', 'media', 'files', ...] // data.version = "1.0.0" // data.count = 80 });
GET/api/icons.json?category=ui

Filter icons by category. Available: ui, media, files, arrows, comms, data, weather, objects.

const res = await fetch('/api/icons.json?category=ui'); const { icons } = await res.json(); // icons filtered to category "ui"
GET/api/icons/{id}.svg

Get a single icon as a raw SVG file by its ID slug.

// Example: /api/icons/home.svg // Returns: Content-Type: image/svg+xml // Use in <img src="..."> or inline fetch
GET/api/search?q={query}

Full-text search across icon names and tags.

const res = await fetch('/api/search?q=arrow'); const { icons, total } = await res.json(); // Matches icons by name, tags, and category
